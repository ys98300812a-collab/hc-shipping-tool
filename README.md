# 🚀  出貨明細自動轉單系統 v3.0 Pro

這是一個專為電商營運與物流流程開發的高效率自動化工具。透過前端技術實現「即傳即轉」，將繁瑣的 Excel 訂單數據轉化為標準化的 A4 PDF 出貨明細單，大幅提升倉庫裝箱與核對效率。

## ✨ 核心價值 (Impact)
- **效率倍增**：將傳統手動複製、調整排版的時間從 10 分鐘縮短至 30 秒。
- **靈活配置**：首創「動態欄位對應」與「個別單據微調」邏輯，適應各類電商平台導出的複雜格式。
- **數據隱私**：100% 本地端運算，數據不經過伺服器，確保企業商業機密安全。

## 🛠 進階功能 (Key Features)

### 1. 智慧分組與自動分頁
系統自動識別 Excel 中的「訂單編號」，將相同訂單的商品自動歸類於同一張出貨單，並實現一單一頁與自動總頁碼標註。

### 2. 客製資訊與拖曳排序 (v3.0 新功能)
- **多樣化來源**：支援從 Excel 自動抓取資料，或手動輸入統一的靜態文字（如：廠商資訊、促銷口號）。
- **可視化排序**：整合 `SortableJS`，使用者可透過拖曳 **⠿** 圖示，自由調整資訊在 PDF 上的顯示順序。

### 3. 個別訂單強力微調 (Individual Override)
針對特定單據的特殊需求（如：補寄贈品、特殊物流備註），支援在預覽區進行「單筆修改」。修改後僅影響該張單據，不連動其他資料，提供極高的彈性。

### 4. 完美排版與批量下載
優化 PDF 渲染邏輯，解決瀏覽器縮放導致的位移問題，確保下載後的 PDF 內容緊湊置頂、不產生空白頁。

## 🔧 技術架構 (Technical Stack)
- **Frontend**: HTML5, CSS3 (Grid & Flexbox layout), Vanilla JavaScript
- **Data Parsing**: [SheetJS](https://sheetjs.com/)
- **PDF Rendering**: [html2pdf.js](https://ekoopmans.github.io/html2pdf.js/)
- **Interactivity**: [SortableJS](https://sortablejs.com/)

## 📖 操作流程 (How to Use)
1. **上傳資料**：拖放 `.xlsx` 或 `.csv` 檔案至上傳區。
2. **欄位對照**：設定 PDF 標題並選擇對應的 Excel 欄位。
3. **客製設定**：新增額外資訊欄位，並利用拖曳功能調整順序。
4. **預覽與修正**：點擊產出預覽。若特定單據需微調，點擊該頁左側的 **📝 修改** 按鈕。
5. **批量下載**：確認無誤後點擊下載，獲取完整 PDF 出貨明細。
